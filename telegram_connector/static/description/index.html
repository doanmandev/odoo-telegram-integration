Telegram Connector for Odoo

Seamless Integration with Telegram Bot API for Real-time Communication

Doan Man - Github

Overview

Telegram Connector is a powerful Odoo module designed to establish bidirectional communication between your Odoo instance and Telegram users through the Telegram Bot API. This module provides a complete webhook-based implementation for receiving and processing messages from Telegram, with message persistence, chat management, and a user-friendly interface to monitor and control your Telegram integrations.

Key Features

  • Webhook Integration: Receives real-time updates from Telegram using webhooks
  • Message History: Records all sent and received Telegram messages for traceability and debugging
  • Multi-Bot Support: Configure and manage multiple Telegram bots from a single interface
  • User-friendly Interface: Control and monitor Telegram communications directly from the Odoo UI
  • Two-way Communication: Both receive messages from and send responses to Telegram users
  • Detailed Message Logs: Comprehensive logging of message details, chat information, and user data

Technical Specifications

  • Webhook Endpoint: /telegram/webhook
  • Authentication: Bot tokens and chat IDs for secure communication
  • Message Direction: Supports both incoming (received) and outgoing (sent) messages
  • Record Keeping: Detailed tracking of message content, user details, and timestamps
  • JSON Processing: Built-in utilities for handling Telegram API JSON responses
  • Dependencies: Base Odoo and requests library

Requirements

For the proper functioning of this module, you need:

  • Requests Library:
    A Python library for making HTTP requests. Usually comes pre-installed with Odoo, but can be installed via pip if needed: pip install requests (Official site: https://docs.python-requests.org/)

  • Telegram Bot:
    You need to create a Telegram bot through BotFather and obtain a bot token. For instructions, visit Telegram Bot Documentation

  • Public-facing Server:
    For webhook functionality, your Odoo server must be accessible from the internet with a valid domain or IP.

Installation & Configuration

  1. Install this module (telegram_connector) through Odoo's app management interface
  2. Create and configure your Telegram bot in the Telegram Configuration menu
  3. Set up the webhook URL in Telegram by calling the Telegram API:
    https://api.telegram.org/bot<BOT_TOKEN>/setWebhook?url=https://yourodoodomain.com/telegram/webhook
  4. Test your configuration by sending a test message using the "Send Message Test" button
  5. Monitor incoming and outgoing messages through the Message Log interface

Use Cases

  • Customer Support: Provide customer service through Telegram messaging
  • Notification System: Send automated notifications about orders, shipments, or other events
  • Lead Generation: Capture leads through Telegram conversations
  • Internal Communication: Connect Odoo with company Telegram groups for internal updates
  • Order Placement: Allow customers to place orders directly through Telegram
  • Status Updates: Provide order status updates or inventory information via chat

Module Structure

  • Telegram Configuration (telegram.config): Manage Telegram bot settings and API tokens
  • Message Log (telegram.message.log): Comprehensive log of all Telegram communications
  • Webhook Controller: Handles incoming webhook requests from Telegram
  • Utility Functions: Helper methods for processing Telegram data and responses

Security Considerations

  • Bot tokens are stored securely in the Odoo database
  • The webhook endpoint requires proper validation of incoming data
  • Access to message logs and configuration is controlled by Odoo's security model
  • We recommend using HTTPS for all communications to ensure secure data transfer
